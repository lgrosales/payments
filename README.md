# Payments API

## Requirements

[.NET Core 2.0.0 SDK](https://www.microsoft.com/net/core)

## How to run

cd /src/Payments.API
dotnet run 
navigate to /swagger
have fun!

## IDE

Can use VS Code or Visual Studio 2017
